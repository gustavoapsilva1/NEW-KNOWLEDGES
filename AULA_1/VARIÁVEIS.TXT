VARIÁVEIS SÃO LOCAIS QUE ARMAZENAM DADOS TEMPORARIAMENTE NA MEMÓERIA RAM. (SERVE PARA MANIPULAR DADOS)

DADOS POSSUEM TIPOS, OS TIPOS PRIMITIVOS, COMUNS A TODAS AS LINGUAGENS DE PROGRAMAÃO SÃO:


NUMÉRICOS:
INTEGER = DADOS NUMÉRICOS DO TIPO INTEIRO: 1, 2, 3, -1, -2, -3 ...

FLOAT = DADOS NUMÉRICOS DO TIPO REAL: -1, 1, 1.5, 2, 2.5, 3, 3.5


LITERAIS:
STRINGS = DADOS DE CARACTER E CADEIA, FICAM ENTRE ASPAS SIMPLES (''), DUPLAS ("") OU TRIPLAS (''')


BOOLEANAS:
TRUE OU FALSE = RETORNO DE OPERAÇÕES REALACIONAIS

PARA CRIAR VARIÁVEIS, PRECISAMOS COLOCAR UM NOME, ATRIBUIÇÃO E UM DADOS

a = 12 // a é o nome, = é a atribuição e 12 é o DADO

EM PYTHON AS VARIÁVEIS SÃO DINÂMICAS, OU SEJA, PODE ACEITAR VÁRIOS TIPOS DE DADOS EM UMA MESMA VARÁVEL.
